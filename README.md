# Multiplication tables

Program to create Multiplication tables in .txt

# To run it:
node app -b (for the base) -l (to the length).

# Example
 node app --base 5 --length 10

 output:
  5 x 1 = 5
  5 x 2 = 10
  5 x 3 = 15
  5 x 4 = 20
  5 x 5 = 25
  5 x 6 = 30
  5 x 7 = 35
  5 x 8 = 40
  5 x 9 = 45
  5 x 10 = 50
